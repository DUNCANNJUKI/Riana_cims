# User Guide

1. Sign in through the RIANA login page. Developer accounts open the Developers workspace automatically.
2. Use the single sidebar to open clients, installations, assignments, reports, feedback, or Developers features available to your role.
3. Notifications in the header cover assignments, account changes, password changes, approvals, completions, and handovers. Developers approval, assignment, and completion events also attempt email and SMS delivery to the responsible system user.
4. Upload signed E-Handover documents from the installation. Preview opens the protected document; Download saves the same verified file.
5. Client Satisfaction is CSAT: the percentage of valid 1–5 responses rated 4 or 5. Average Rating and NPS are displayed separately.
6. The session warns before ten minutes of inactivity. Continue to remain signed in or sign in again after automatic logout.

For a problem, open Help, search the knowledge articles, or use the assistant. Never send passwords, reset tokens, or client documents through chat.

Help & Support provides role-specific recommendations, searchable and expandable workflow articles, the RIANA Assistant, a support guide delivered to your signed-in work email, system requirements, PWA installation guidance, security guidance, and approved RIANA Support/System Administrator/Live User Chat channels.

## Roles, Pending, and reports

- Finance can view Clients, Installations, Assigned Technicians, Installation Progress, and all Reports without edit controls.
- Management can administer the system but cannot add, edit, update, or change progress on installations.
- Sales can view all Reports.
- Super Admin can edit a user and tick Individual Extra Privileges. Role-included privileges are locked; Management installation restrictions cannot be overridden.
- Developers > Pending shows approvals, unassigned work, clarification requests, and assigned work requiring action. Developers Reports can be filtered by developer name.
- Dashboard activity stays hidden until Recents is selected.
- E-Handover equipment with quantity `0` is labelled `Not installed` in preview and PDF.
